Adds nofollow tag to links in your site's content on the basis of categories, tags, post types and users.

== Description ==
This plugins gives you complete control over your website's links and pagerank. It can help you nofollow links in your site's content on the basis of categories, tags, post types and users.

In the control panel you can specify the categories, tags, post types and users for which you want to nofollow links and the plugin will take care of the rest.

If you like this plugin please give it a good rating.

== Installation ==
1. Just install the plugin using Wordpress' default plugin installer and activate.
2. Go to the control panel listed under Plugins and set the categories, tags and users for which you want to nofollow the links.
3. That's it! Enjoy!
